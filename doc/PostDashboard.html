COLLECTION_FILTERS =

COLLECTION_FILTERS a hash that defines filters that can be used while searching via the search field of the dashboard.

For example to add an option to search for open resources by typing “open:” in the search field:

COLLECTION_FILTERS = { open: ->(resources) { resources.where(open: true) } }.freeze
